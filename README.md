# 🚀 AllianzAmerica

Banner AllianzAmerica

## 🌎 Sobre o Projeto

A **AllianzAmerica** é uma landing page moderna desenvolvida com foco em tecnologia, inovação e aprendizado digital. O projeto foi criado para representar uma plataforma de cursos e desenvolvimento profissional voltada para estudantes, programadores e pessoas apaixonadas por tecnologia.

A proposta da plataforma é oferecer uma experiência visual moderna, intuitiva e profissional, utilizando conceitos atuais de UI/UX Design, responsividade e estilização avançada com CSS.

---

# ✨ Funcionalidades

✅ Landing Page moderna
✅ Design responsivo
✅ Cards de cursos interativos
✅ Sessão Sobre Nós
✅ Footer profissional
✅ Ícones com Font Awesome
